How to Use Devstar+

Learn how to explore the Devstar+ portfolio experience, interact with cinematic UI, and navigate through modern OTT-style components.

01

Browse Content

Scroll through sliders to explore different UI sections like movies, categories and featured content just like OTT platforms.

02

Hover Interaction

Hover on cards to see preview animations, details, and smooth cinematic UI effects built with modern frontend techniques.

03

Watch Preview

Cards support video previews. Hover triggers playback for an immersive preview experience similar to Netflix & Hotstar UI.

04

Open Modal

Click "Watch Now" to open full preview modal with video player, description, tags and cinematic layout design.

05

Navigation

Use header menu to switch between sections like Home, About, Careers and other pages easily.

06

Responsive Design

The entire UI is optimized for mobile, tablet and desktop ensuring smooth experience across all devices.

Note

Devstar+ is a creative portfolio project designed to showcase modern UI/UX skills. All content is for demonstration purposes only and built using HTML, CSS and JavaScript.
